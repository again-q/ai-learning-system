# K 维度最终方案（全）

## 1. 知识点结构
- **三重结构**：知识层（计入K）、模型层（参考）、方法层（参考）
- **七要素归属**：定义、性质、操作、关系、表示 → K；应用 → A；来源 → Q
- **要素追踪**：同一话题下各要素独立追踪 S_k/D_k，不合并

## 2. 掌握度计算（加权得分法）

```
每道题贡献 = 难度(0.01-0.99) × 过程分(0/0.3/0.5/1.0)
知识点累计得分 S_k = Σ 贡献
知识点累计难度 D_k = Σ 难度
掌握度 = S_k / D_k
```

## 3. 难度（LLM判定）
- 5维度评估：知识复杂度、思维深度、陷阱密度、计算强度、陌生度
- 每个维度 0-1，加权平均 → 综合难度 0.01-0.99
- LLM 自行发挥，不锁死档位

## 4. 过程分（LLM判定）
- 1.0 → 完全正确 + 思路清晰
- 0.5 → 部分正确 / 正确但思路模糊
- 0.3 → 错误但有正确思路
- 0 → 完全错误 / 空白

## 5. 4档展示
- ≥75% → 🟢 掌握
- 35-75% → 🟡 巩固
- 10-35% → 🟠 薄弱
- <10% → 🔴 紧急

## 6. 超纲规则
- LLM 判断是否超纲
- 超纲且做错 → 不降掌握度（不计入 D_k）
- 超纲且做对 → 正常计入

## 7. 遗忘机制
- 超过14天未练习 → 触发遗忘
- 有 Q 维度"理解标记"（思维链判定 ≥0.7）→ 衰减系数 0.5x
- 无理解标记 → 衰减系数 1.0x
- 每周降 5% 档位，不降 S_k/D_k

## 8. 学习进度指标（未来）
- 只增不减，与掌握度互补
