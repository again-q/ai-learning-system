# AI 学习助手 — 开发路线图

> 本文档记录项目的开发方向、优先级和进度。  
> **每次讨论产生新功能、调整优先级或完成开发，必须同步更新本文档。**
>
> 项目包含两条线：
> - **子项目**：AI 智能体小程序（当前优先，练手 + 攒 AI 基础设施）
> - **主程序**：薄弱点诊断工具（子项目成熟后继续）

---

## 🔴 子项目：AI 智能体小程序（当前优先）

> 一个可自定义角色的 AI 对话助手，支持持久记忆。在主项目内独立目录开发，代码最终可复用到主程序 AI 模块。
>
> **模型**：DeepSeek | **技术栈**：微信云开发 | **盈利**：广告（非当前重点）

- [ ] **环境搭建** — 新建目录、云函数骨架、DeepSeek API 接入
- [ ] **对话 MVP** — 单助手 + 可自定义角色设定 + 持久记忆
- [ ] **多对话管理** — 支持创建/切换/删除多个对话会话
- [ ] **记忆系统** — AI 能记住关键信息跨对话延续
- [ ] **主程序复用** — 将 AI 对话模块抽取为可复用组件

---

## 🟢 Phase 1：诊断闭环（主程序）

> 先跑通「录入 → 诊断 → 反馈」的最小闭环，聚焦数学单科。
> 核心原则：录入必须极轻量，否则没人用。

- [ ] **拍照录入 MVP** — 拍照 → AI 识别题目/作答 → 学生填正确答案 → AI 关联知识点
- [ ] **知识图谱搭建（数学）** — 初中数学知识点 DAG，前置/后继关系
- [ ] **K 维度评分引擎** — 基于 BKT 的知识点掌握度计算
- [ ] **诊断输出** — 结构化报告：薄弱点定位 + 认知科学依据 + 行动计划
- [ ] **定点突破练习** — 针对薄弱知识点的轻量练习

---

## 🟡 Phase 2：诊断深化

> 五维模型全面落地，诊断精度提升。

- [ ] **A 维度** — IRT 能力水平估算（综合卷/模拟考数据）
- [ ] **T 维度** — 迁移能力分析（变形题识别）
- [ ] **Q 维度** — 思维品质分析（思维链 + 反思数据）
- [ ] **S 维度** — 执行稳定分析（考试 vs 日常对比）
- [ ] **AI 学习计划** — 根据五维画像动态生成学习路径

---

## 🔵 Phase 3：体验打磨

> 核心闭环跑通后，全面优化 UI 与交互。

- [ ] **UI 全面翻新** — 按设计语言规范 v1.1 统一视觉
- [ ] **动画与微交互** — 入场动效、状态切换、加载反馈
- [ ] **录入体验优化** — 整页拍拆题、批量录入
- [ ] **知识图谱可视化** — 知识点掌握度热力图
