# Task: 五维能力模型理论完善

## Goal
完善五维能力模型（K/A/T/Q/S）的定义、测量方式、计算逻辑，使其可工程落地。

## Scope
- K: BKT + 遗忘曲线 (基本就绪)
- A: 懂→会用的能力 (定义就绪，算法待定)
- T: 跨知识点迁移能力 (定义就绪，算法待定)
- Q: 从错误中学习的能力 (定义讨论中，核心难点)
- S: 执行稳定性 (未开始)

## Non-Goals
- 具体代码实现
- UI/UX 设计
- 知识图谱搭建
- 拍照上传流程

## Key Constraints
- 只能从 7 项数据中提取：原图、AI识别文字、学生作答、正确答案、对错、思维链(可选)、时间戳
- 不能增加学生操作负担
- 轻量化优先
