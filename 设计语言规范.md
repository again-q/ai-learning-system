# AI 学习助手 · 设计语言规范 v1.1

> 美学方向：**Apple/Notion 极简白色简约**
> 适用范围：微信小程序前端（WXML/WXSS）
> 制定日期：2026-07-02
> 强制级别：**所有新页面与视觉迭代必须遵循本规范**
> 修订：v1.1 改为白色简约方向（原 v1.0 深色方向已废弃）

---

## 一、美学理念

### 1.1 设计概念

**「克制的留白」**

Apple/Notion 式极简——大量留白、克制的色彩、精致的字重对比、丝滑的微交互。不堆砌装饰，靠**间距、字重、动画**三件事撑起质感。

### 1.2 设计原则

1. **白底为主**：背景永远是白色 `#ffffff`，分区用 `#f5f5f7` 浅灰
2. **单一主色**：蓝色 `#007aff` 是唯一强调色，每屏只允许一个蓝色焦点
3. **字重对比**：标题 700、副标题 600、正文 400——靠字重而非颜色制造层次
4. **动画克制**：动效要丝滑但不过度，重点是微交互反馈和入场过渡

### 1.3 反对的设计

- ❌ 深色背景、彩色光晕、噪点纹理（v1.0 方向已废弃）
- ❌ 多色彩竞争（每屏不超过 2 个色相）
- ❌ 过度装饰（引号、徽标、光晕、光泽流动）
- ❌ 阴影过重（深色背景才需要重阴影，白底用轻阴影）

## 二、色彩系统

```css
/* ========== 背景 ========== */
--bg:            #ffffff;   /* 主背景 */
--bg-subtle:     #f5f5f7;   /* 次背景（卡片底、禁用态） */
--bg-input:      #f5f5f7;   /* 输入框底 */

/* ========== 文字 ========== */
--text-primary:   #1d1d1f;  /* 主文字 */
--text-secondary: #6e6e73;  /* 次文字 */
--text-tertiary:  #a1a1a6;  /* 弱文字（占位/禁用） */

/* ========== 强调色 ========== */
--accent:        #007aff;   /* 主色（iOS 蓝） */
--accent-soft:   #4a9eff;   /* 浅蓝（hover） */
--accent-bg:     rgba(0,122,255,0.06);  /* 蓝色背景（选中态） */

/* ========== 五维能力色（仅图谱页用） ========== */
--dim-k:  #007aff;
--dim-a:  #34c759;
--dim-t:  #ff9500;
--dim-q:  #af52de;
--dim-s:  #ff3b30;

/* ========== 边框 ========== */
--border:      #d2d2d7;     /* 主边框 */
--border-soft: #e5e5ea;     /* 弱边框 */

/* ========== 阴影（白底用轻阴影） ========== */
--shadow-sm: 0 1rpx 2rpx rgba(0,0,0,0.04);
--shadow-md: 0 4rpx 16rpx rgba(0,0,0,0.06);
--shadow-lg: 0 8rpx 32rpx rgba(0,122,255,0.18);  /* 蓝色按钮专用 */
```

### 色彩使用原则

1. **白底为主**：90% 页面是白底，灰底仅用于卡片区分
2. **单一强调色**：每屏只有一个蓝色焦点（按钮或关键数字）
3. **五维色仅图谱用**：其他页面不得使用五维色

## 三、字体系统

```css
--font-sans:    -apple-system, "PingFang SC", "Helvetica Neue", sans-serif;
--font-mono:    "SF Mono", monospace;  /* 仅数字统计用 */
```

### 字阶

| Token | 字号 | 字重 | 用途 |
|-------|------|------|------|
| display | 56rpx | 700 | 关键大数字 |
| h1 | 44rpx | 600 | 页面主标题 |
| h2 | 36rpx | 600 | 区块标题 |
| h3 | 32rpx | 600 | 卡片标题 |
| body | 28rpx | 400 | 正文 |
| body-lg | 30rpx | 500 | 重要正文 |
| caption | 24rpx | 400 | 说明文字 |
| micro | 22rpx | 400 | 极小说明 |

**原则**：标题用 600（不用 700，白底下 600 已足够有力），正文用 400。

## 四、间距系统（8 倍数）

```css
--space-xs:  8rpx;
--space-sm:  16rpx;
--space-md:  24rpx;
--space-lg:  32rpx;
--space-xl:  48rpx;
--space-2xl: 64rpx;
--space-3xl: 80rpx;
```

| 场景 | 间距 |
|------|------|
| 页面左右边距 | 48rpx |
| 卡片内边距 | 32rpx |
| 区块间垂直间距 | 48rpx |
| 同卡片内元素 | 24rpx |
| 标题与副标题 | 8rpx |

**对齐原则**：所有居中元素必须用 `display:flex; align-items:center`，不靠 padding 估算。

## 六、动画规范（重点）

### 6.1 缓动函数

```css
--ease-out:    cubic-bezier(0.16, 1, 0.3, 1);    /* 主缓动 */
--ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1); /* 弹性（按压回弹） */
```

### 6.2 时长

| 场景 | 时长 |
|------|------|
| 按钮按压 | 200ms |
| 卡片/元素按压 | 300ms |
| 输入框聚焦线扩展 | 400ms |
| 入场动画 | 600ms |

### 6.3 入场动画（staggered reveal）

```css
@keyframes reveal-up {
  from { opacity: 0; transform: translateY(20rpx); }
  to   { opacity: 1; transform: translateY(0); }
}
.reveal { animation: reveal-up 0.6s var(--ease-out) both; }
.reveal-1 { animation-delay: 0.10s; }
.reveal-2 { animation-delay: 0.22s; }
.reveal-3 { animation-delay: 0.34s; }
```

### 6.4 微交互反馈（重点）

| 元素 | 反馈 |
|------|------|
| 按钮 | `transform: scale(0.98)` + `opacity: 0.9` |
| 卡片 | `transform: scale(0.96)` |
| 头像 | `transform: scale(0.94)` |
| 输入框聚焦 | 下划线从 64rpx 扩展到 100% + 颜色变蓝 |
| 品牌标志 | 按压时 `scale(0.92)` 弹性回弹 |

### 6.5 禁用的动画

- ❌ 光泽流动（`shine` 关键帧）
- ❌ 光晕脉动（`glow-pulse`）
- ❌ 呼吸闪烁（`dot-blink`）
- ❌ 模糊渐入（`filter: blur`）

**简约风的动画原则**：只在用户操作时给反馈，不做无意义的循环动画。

## 七、组件规范

### 7.1 主按钮

```css
.btn-primary {
  background: var(--accent);
  color: #fff;
  height: 96rpx;
  border-radius: 48rpx;
  box-shadow: var(--shadow-lg);  /* 蓝色光晕阴影 */
  transition: all 0.2s var(--ease-out);
}
.btn-primary:active {
  transform: scale(0.98);
  opacity: 0.9;
}
.btn-primary[disabled] {
  background: var(--bg-subtle);
  color: var(--text-tertiary);
  box-shadow: none;
}
```

### 7.2 卡片

```css
.card {
  background: #fff;
  border-radius: 24rpx;
  padding: 32rpx;
  box-shadow: var(--shadow-md);
}
```

### 7.3 输入框

```css
.input {
  background: transparent;
  border-bottom: 2rpx solid var(--border);
  padding: 20rpx 0;
}
.input:focus {
  border-color: var(--accent);
}
```

## 八、与 reasonix 的协作约定

1. **白底**：所有页面背景必须为 `#ffffff`，禁止深色背景
2. **主色**：用 `#007aff`，禁止琥珀色、紫色等其他主色
3. **阴影**：白底用轻阴影 `rgba(0,0,0,0.04~0.06)`，禁止重阴影
4. **动画**：只做微交互反馈和入场过渡，禁止循环装饰动画
5. **装饰**：禁止引号、光晕、噪点等装饰元素
6. **字号**：标题用 600 字重（不用 700）
