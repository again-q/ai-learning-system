# 五维能力模型 — 学科掌握诊断框架

> 理论构建：林圻  
> 版本：v2.0（2026-07-09）  
> 状态：理论定稿，工程定义已细化  
> 定位：**课后辅助诊断工具，不代替学校**

---

## 一、核心命题

用一个可量化的、可操作的模型，让一个初高中生知道自己"到底哪里不会"、"为什么不会"、"下一步该做什么"。

三个子问题：
1. **"掌握"到底指什么？**—— 不只是"会不会"，而是"会到什么程度"
2. **怎么把它变成可测的数字？** —— 从学生日常做题数据中提取五个维度的信号
3. **测出来之后怎么用？** —— 输出可操作的诊断报告和行动建议

---

## 二、核心 Pipeline

所有维度的计算基于统一的数据管道：

```
拍照 → AI 识别题目 + 学生作答 → 对比正确答案
    ↓
LLM 错因分类（将"错"拆为 4 类）
    ↓
计算失误/抄错数字 → S 维度扣分
套错公式但结构对   → A 维度扣分
题意理解偏/答非所问 → T/K 维度扣分
完全空白/乱填      → K 维度扣分
    ↓
    高价值题目？ → 弹窗问思维链 → Q 评分
    ↓
更新五维分数 → 输出诊断报告
```

**核心原则**：每个维度使用互不重叠的数据源，避免维度间的信号污染。

---

## 三、五维能力向量

```
M = (K, A, T, Q, S)
```

| 维度 | 对学生说 | 认知本质 | 数据源 | 覆盖率 |
|------|---------|---------|--------|--------|
| K | 这个知识点你学过吗？ | 长时记忆中的知识节点 | 难度×过程分（加权得分法） | 全量 |
| A | 学会了能不能用出来？ | 同知识点跨题型运用 | LLM区分概念/应用题 | 全量 |
| T | 不同知识能串起来吗？ | 跨知识点聚合 | 知识图谱DAG | 依赖图谱 |
| Q | 知道自己怎么错的吗？ | 自我觉察与调整 | 思维链（弹窗触发） | 稀疏 |
| S | 发挥稳不稳定？ | 预期vs实际偏差 | K/A/T综合预期对比 | 全量 |

---

### K — 知识掌握

**本质**：知识点在你的长时记忆里存好了吗？

**通俗说法**："知不知道"

**三层结构**：
| 层 | 内容 | 计入K总分 |
|---|------|---------|
| 知识层 | 概念定义、教材方法 | ✅ 是 |
| 模型层 | 题目模型、解题框架 | ❌ 参考 |
| 方法层 | 思维技巧、解题策略 | ❌ 参考 |

**算法**：加权得分法（S_k / D_k）

**基本公式**：
```
每道题贡献 = 难度(0.01-0.99) × 过程分(0/0.3/0.5/1.0)
知识点累计得分 S_k = Σ 贡献
知识点累计难度 D_k = Σ 难度
掌握度 = S_k / D_k
```

**难度判定（LLM，5维度加权平均）**：
| 维度 | 低分 | 高分 |
|------|------|------|
| 知识复杂度 | 单步套公式 | 多步+跨知识点 |
| 思维深度 | 直接套用 | 逆向/构造 |
| 陷阱密度 | 直白无坑 | 多陷阱/分类讨论 |
| 计算强度 | 口算级别 | 复杂运算 |
| 陌生度 | 常规题 | 新题型 |

**过程分判定（LLM）**：
- 1.0 → 完全正确，思路清晰
- 0.5 → 部分正确 / 正确但思路模糊
- 0.3 → 错误但有正确思路
- 0 → 完全错误 / 空白

**超纲规则**：LLM 判定超纲的题目，做错时不降掌握度（不计入 D_k）
**遗忘机制**：超过 14 天未练习触发，有 Q 维度理解标记的衰减减半
**展示**：4 档（掌握/巩固/薄弱/紧急） + 学习进度指标（待实现）

**七要素归属**：
| 要素 | 归属维度 |
|------|---------|
| 定义、性质、操作、关系、表示 | K 维度 |
| 应用 | A 维度 |
| 来源 | Q 维度 |

---

### A — 能力水平

**本质**：从"知道"到"会用"的距离 —— 识别出知识点并运用它解题

**通俗说法**："会不会用"

**核心区别**：
- K = 你知不知道这个公式
- A = 你知不知道什么时候该用它

**算法**：IRT 1PL + 跨题型对比
- P(做对|能力θ, 难度b) = 1/(1+e^-(θ-b))
- A = 应用正确率 / 概念正确率（同一知识点下）

**数据来源**：LLM 自动识别题目类型（概念题 vs 应用题），对比正确率差距（全量）

---

### T — 迁移能力

**本质**：不同知识点之间的触类旁通——把不同知识串起来解决复杂问题

**通俗说法**："触类旁通"

**递进关系**：
```
K = 一个知识点"知不知道"
A = 一个知识点"会不会用"（同知识点不同场景）
T = 多个知识点"能不能串起来"（跨知识点聚合）
```

**算法**：基于知识图谱 DAG 的跨节点表现
- 正迁移：连续做对两个关联知识点 → T 上升
- 负迁移：错误地将 B 知识点公式用到 A 知识点 → T 下降

**前提**：依赖学科知识图谱的搭建

---

### Q — 思维品质

**本质**：觉察和调整的能力 —— 你知不知道你怎么错的，下次能不能改过来

**通俗说法**："会不会学"

**核心原则**：
- Q 只通过思维链测量。没有思维链，Q 不更新。
- 思维链不让学生主动写，而是 AI 判断为高价值题目后弹窗询问
- 对题也能涨 Q（思路清晰说明觉察能力强）

**触发方式**：AI 识别高价值题目 → 弹窗问"你是怎么想的？"
- 高价值判断：做错的题、做对但过程复杂的题、涉及关键知识节点的题
- Q 低 → 每次高价值题都弹窗
- Q 高 → 减弹窗，鼓励主动写

**评分方式（有思维链时）**：
```
LLM 评思维链的三个维度：
① 归因准确度 — 是否准确说出了错因/思路？
② 调整清晰度 — 是否说了下次怎么改？
③ 执行结果   — 下次同类题是否做对了？

对题：思路清晰 → Q +
错题：归因准 + 调整明确 → Q ++
无思维链 → Q 不更新（维持原值或微幅衰减）
```

**数据来源**：思维链（弹窗触发，稀疏但独占）

**理论依据**：思维链是 Q 的唯一可观测窗口。自我觉察是内在心理过程，只能通过外在的语言表达（思维链）来推断。没有思维链时，我们能观测的只有行为结果（对/错），但那已经被 K/A/T 覆盖了。

---

### S — 执行稳定

**本质**：预期偏差 —— 你实际发挥相对于你应该发挥水平的偏离程度

**通俗说法**："稳不稳定"、"明明会为什么做错了？"

**认知本质**：预期 vs 实际的偏差
- 低于预期（失常）→ S 下降
- 高于预期（超常）→ S 微涨（权重更低，因为可能是偶然）

**算法**：
```
预期值   = 由 K/A/T 综合算出的"这道题你应该做对的概率"
单次偏差 = 实际结果(1/0) - 预期概率
S       = 过去 N 次偏差的加权移动平均
          （失常权重 1.0，超常权重 0.3）
```

**数据来源**：K/A/T 综合预期 vs 实际对错结果（全量）

**与错因分类的关系**：计算失误/抄错数字是 S 维度的主要负向信号，但不是唯一信号。

## 四、各维度数据源独立性（非重叠原则）

### 核心原则

**每个维度有且只有自己独占的信号源，一个信号不能同时用于计算两个维度。**

### 当前确认的信号分配

| 维度 | 独占信号 | 不使用的信号 |
|------|---------|------------|
| K | 难度×过程分 | 无（加权得分法0外部参数） |
| A | 概念题 vs 应用题的正确率差距 | 不直接使用对/错结果 |
| T | 跨知识点的关联作答模式 | 不单独处理单点对错 |
| Q | 思维链文本（弹窗触发） | 不自行分析对错结果 |
| S | 实际表现 vs K/A/T 预期偏差 | 不自行判断"这是粗心还是不会" |

### 跨维度参数传递

K 维度的加权得分法不需要外部参数。

```
K 的计算只依赖于：
  难度 0.01-0.99 ← LLM 判定（5维度加权）
  过程分 0-1    ← LLM 判定（4档）
  
  不需要 P(S)、P(G)、P(L₀)、P(T) 等参数
```

K 向其他维度传递的信号：
```
- 具体到题目的过程分和思维链 → 提供给 Q 维度
- 超纲/不超纲标记 → 提供给学习进度指标
```

```
❌ K 的遗忘衰减依赖 Q 维度的理解标记 → 但这是协同关系，不是信号重叠
❌ Q 在掌握度序列上做回归，K 也在同一序列上算掌握度 → 两个维度用同一数据
✅ 解决方案：Q 只看思维链，K 只看对错+难度+过程分，互不干扰
```

## 五、认知科学依据

| 维度 | 唯一数据 | 不依赖 | 覆盖率 |
|------|---------|--------|--------|
| K | 难度×过程分 | 无（无外部参数） | 全量 |
| A | 概念题 vs 应用题分类 | LLM 分类能力 | 全量 |
| T | 知识图谱跨节点关联 | 图谱搭建完成度 | 条件 |
| Q | 思维链文本 | 学生写思维链的意愿 | 稀疏 |
| S | K/A/T 综合预期 vs 实际 | 其他四个维度的准确度 | 全量 |

**五维不重叠**：每个维度有自己独占的信号源，不存在两个维度用同一数据算不同分的情况。

## 六、适配层次

| 层次 | 学段 | 推荐维度 | 重心 |
|------|------|---------|------|
| 筑基期 | 高一 | K 主导 | 扫清知识盲区 |
| 连接期 | 高二 | K + A + T | 建立知识网络 |
| 兑现期 | 高三 | A + S 主导 | 稳定发挥能力 |

**核心观点**：同一个学生不同阶段的重心不同，五维模型动态适配。

---

## 七、一句话总结

> 一个拍照上传题目就能出诊断报告的学习分析工具。不搜题、不讲课、不刷题——只做一件事：**告诉你到底哪里不会，为什么不会，然后怎么学。**
