## 一、核心命题

用一个可量化的、可操作的模型，刻画一个人在某一学科上的"真实掌握程度"，并指导他走向高水平。

三个子问题：
1. **"掌握"到底指什么？**（认知分层）
2. **怎么把它变成可测的数字？**（量化引擎）
3. **测出来之后怎么用？**（实践闭环）

---

## 二、认知分层（布鲁姆分类学本土化改造）

| 层级 | 名称 | 含义 | 对应实践动作 |
|------|------|------|------------|
| L1 | 事实概念层（know-what） | 术语、定义、经典结论 | 概念记忆 |
| L2 | 结构脉络层（know-why/关联） | 概念间的推导、因果、分类从属关系 | 关联梳理 |
| L3 | 程序迁移层（know-how+创造） | 用学科工具解决陌生问题 | 迁移应用 |

**核心观点**：只测L1叫"背诵"，只练L3叫"刷题"，真正的掌握是三层贯通。

## 三、量化引擎：五维能力向量

```
M = (K, A, T, Q, S)
```

### 核心 Pipeline（工程实现路径）

所有维度的计算基于统一的数据管道：

```
拍照 → AI 识别题目 + 学生作答 → 对比正确答案
    ↓
LLM 错因分类（将"错"拆为 4 类）
    ↓
计算失误/抄错数字 → S 维度扣分
套错公式但结构对   → A 维度扣分
题意理解偏/答非所问 → T/K 维度扣分
完全空白/乱填      → K 维度扣分
    ↓
更新五维分数 → 输出诊断报告
```

### K — 知识掌握（Knowledge Mastery）

- **定义**：某个知识点在你的长时记忆里存好了吗？
- **通俗说法**："知不知道"
- **模型**：BKT（贝叶斯知识追踪）+ Ebbinghaus 遗忘曲线
- **公式**（每做完一题更新）：
  - P(掌握|做对) = P(L) × (1-P(S)) / [P(L)×(1-P(S)) + (1-P(L))×P(G)]
  - P(掌握|做错) = P(L) × P(S) / [P(L)×P(S) + (1-P(L))×(1-P(G))]
  - 参数：P(L₀)初始掌握率、P(T)学习率、P(G)猜对率、P(S)失误率
- **遗忘曲线**：掌握度(t) = 掌握度(刚学时) × e^(-t/τ)，τ 随连续做对次数增大
- **数据来源**：同一知识点连续作答的对/错序列 + 时间戳
- **输出**：每个知识节点的掌握概率 p∈[0,1]，按知识图谱加权聚合全局K分数

### A — 能力水平（Ability Level）

- **定义**：从"知道"到"会用"的能力 —— 识别出知识点并运用它解题
- **通俗说法**："会不会用"
- **认知本质**：同知识点内，直问（概念题）与变形（应用题）的表现差距
- **模型**：IRT 1PL（Rasch模型）+ 跨题型对比
- **公式**：
  - P(做对|能力θ, 难度b) = 1/(1+e^-(θ-b))
  - A = f(应用正确率 / 概念正确率)
- **数据来源**：LLM 识别题目类型（概念题 vs 应用题），对比学生在同一知识点上两种题型的正确率差距
- **K 与 A 的区别**：K=你知不知道这个公式，A=你知不知道什么时候该用它

### T — 迁移能力（Transfer Ability）

- **定义**：不同知识点之间的触类旁通能力
- **通俗说法**："触类旁通"
- **认知本质**：跨知识点的聚合能力 —— 不同知识点、不同场景间建立联系
- **K-A-T 的递进关系**:
  - K = 一个知识点"知不知道"
  - A = 一个知识点"会不会用"（同知识点不同场景）
  - T = 多个知识点"能不能串起来"（跨知识点聚合）
- **量化**：基于知识图谱 DAG 的跨节点表现
  - 正迁移：连续做对两个关联知识点 → T 上升
  - 负迁移：错误地将 B 知识点公式用到 A 知识点 → T 下降
- **数据来源**：知识图谱关联网络 + 跨知识点作答序列
- **前提**：依赖学科知识图谱的搭建

### Q — 思维品质（Thinking Quality）

- **定义**：觉察和调整的能力 —— 你知不知道你怎么错的，下次能不能改过来
- **通俗说法**："会不会学"
- **认知本质**：自我觉察能力的外在体现，只能通过思维链观测
- **与 K/A/T/S 的关系**：Q 的提升能带动其他四个维度（共鸣效应），但共鸣是结果不是 Q 本身
- **触发方式**：AI 识别高价值题目 → 弹窗问"你是怎么想的？"（不是学生主动写）
  - 高价值判断规则：做错的题、做对但过程复杂的题、涉及关键知识节点的题
  - Q 低 → 每次高价值题都弹窗；Q 高 → 减弹窗，鼓励主动写
- **评分方式（有思维链时）**：
  - LLM 评思维链的三个维度：
    1. 归因准确度（是否准确说出了错因/思路？）
    2. 调整清晰度（是否说了下次怎么改？）
    3. 执行结果（下次同类题是否做对了？）
  - 对题：思路清晰 → Q +
  - 错题：归因准 + 调整明确 → Q ++
- **无思维链时**：Q 不更新（维持原值或微幅衰减）
- **数据来源**：思维链（弹窗触发，稀疏但独占）
- **理论依据**：思维链是 Q 的唯一可观测窗口。没有思维链就无法测量"觉察"，Q 与其他四个维度的本质区别就是它看过程而非结果

### S — 执行稳定（Stability）

- **定义**：预期偏差 —— 你实际发挥相对于你应该发挥水平的偏离程度
- **通俗说法**："稳不稳定"
- **通俗说法**："明明会为什么做错了？"
- **认知本质**：预期 vs 实际的偏差
  - 低于预期（失常）→ S 下降
  - 高于预期（超常）→ S 微涨（权重更低，因为可能是偶然）
- **公式**：
  - 预期值 = 由 K/A/T 综合算出的"这道题你应该做对的概率"
  - 单次偏差 = 实际结果(1/0) - 预期概率
  - S = 过去 N 次偏差的加权移动平均（失常权重 1.0，超常权重 0.3）
- **数据来源**：K/A/T 综合预期 vs 实际对错结果
- **与错因分类的关系**：计算失误/抄错数字是这个维度的主要负向信号，但不是唯一信号

### 隐性维度：P — 学科品味
- **定义**：对解法优劣的敏感性（"审美"）
- **量化**：解法对比测试 + 自主优化率
- **地位**：不纳入日常雷达图，作为长期成长见证指标

## 四、全科拓展：Core + Adapter 架构

| 差异点 | 数理化（硬学科） | 语文英语（半软学科） | 政史地（文史学科） |
|--------|----------------|-------------------|-----------------|
| 掌握的定义 | 离散、有对错、有推导链 | 连续频谱、无绝对对错 | 因果/叙事、弱依赖 |
| 推荐模型 | BKT（二值隐状态） | IRT + 连续覆盖度 | BKT调参 + 弱约束图谱 |
| 难度标定 | 认知负荷（步骤数+抽象度） | 文本复杂度+题型组合 | 材料陌生度+史观要求 |
| 图谱边语义 | 推导依赖（硬约束） | 语法层级（弱约束） | 因果/时间（中等约束） |

**结论**：元结构全科通用，但"认知原语"每科必须重写。

## 五、适配层次

| 学生层次 | 目标分数 | 推荐维度 | 系统形态 |
|---------|---------|---------|---------|
| 学困生 | <110 | K单维+遗忘曲线 | 错题本Pro版，只做基础巩固 |
| 中等生 | 110~130 | K+A+基础T | 诊断+提分工具 |
| 优等生 | 130~150 | 完整五维+P | 认知增强系统，全流程 |

**同一学生不同阶段的重心**：
- 高一：K筑基（扫盲员）
- 高二：K+T连接（教练）
- 高三：A+S兑现（陪考+策略顾问）
