## 一、AI/LLM 相关

- [ ] **🔴 LLM 错因分类 prompt 设计**
  - 输入：题目原文、学生作答、正确答案
  - 输出：{错误类型, 置信度, 错误步骤定位}
  - 需要：prompt 模板、few-shot 示例、质量抽检方案

- [ ] **🟡 拍照 OCR / 题目识别**
  - 用哪个视觉模型（混元视觉/GPT-4o/Claude）？
  - 识别内容：题目文字、学生作答、手写草稿
  - 置信度阈值、识别失败兜底

- [ ] **🟡 智能私教提示词设计**
  - 用户说"到时候要好好设计"
  - 角色设定、输出格式、知识边界、防幻觉

- [ ] **🟡 Q 维度：思维链评分 prompt**
  - 输入：思维链文本、题目信息、对错结果
  - 输出：归因准确度(0-1)、调整清晰度(0-1)
  - 评分标准定义

- [ ] **🟡 Q 维度：高价值题目判断规则**
  - 什么算高价值？做错的题？做对但复杂的题？
  - 具体判断逻辑（LLM 判断还是规则判断？）

## 二、五维模型工程规格（每个维度展开成完整算法）

- [ ] **🔴 K 维度：BKT 参数初始化**

  - 遗忘曲线 τ 的初始值？
  - 冷启动策略（新学生没有历史数据时怎么算？）

- [ ] **🔴 A 维度：题型分类与 IRT**

- [ ] **🟡 T 维度：知识图谱结构**
  - 数学知识点 DAG 的具体节点和边
  - 谁来建？你先画骨架还是 AI 辅助？
  - 正迁移/负迁移的具体判定逻辑

- [ ] **🟡 Q 维度：评分算法实现**
  - 思维链评分 → Q 值更新的映射函数
  - 无思维链时的衰减策略
  - 弹窗频率控制

- [ ] **🟡 S 维度：预期偏差计算**
  - K/A/T 综合预期的具体公式
  - 滑动窗口大小 N
  - 失常/超常权重的具体值
